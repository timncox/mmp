Add the MCP server

Paste mmp.chat/mcp into your AI's MCP server settings. Works with Claude Desktop, ChatGPT, VS Code Copilot, and any MCP-capable client.

2

Pick a handle

Tell your AI: "Register as @yourname" — that's it. Your encryption keys are generated automatically and your recovery code is saved to your AI's memory.

"Send @alice hello!" — your AI handles encryption, delivery, and threading. Or say "Open my inbox" for a visual interface.

Built for AI, secured for humans

🔒

End-to-end encrypted

Messages are encrypted with NaCl box (x25519). The server never sees your plaintext. Your AI reads it — nobody else.

🌐

Works everywhere

Claude, ChatGPT, Copilot, Goose, Cursor — any AI that supports MCP. One protocol, every client.

🤖

AI-native messaging

Your AI summarizes, prioritizes, and drafts replies. "Give me a digest" gets you a prioritized summary across all threads.

📬

Interactive inbox

On supported clients, a rich inbox UI renders right inside your conversation. Threads, compose, contacts — no app needed.

🙋

Invite anyone

"Invite @bob" generates a shareable link. They add the MCP server, register, and your pending message gets delivered.

🔌

Open protocol

MMP is an open spec. Anyone can run a server. Anyone can build a client. No vendor lock-in, no platform dependency.

For developers

MMP is an open extension to the Model Context Protocol. The spec defines 22 MCP tools, a threading model, hybrid encryption, and an MCP App for interactive inbox UI. Run your own server or build on the protocol.
